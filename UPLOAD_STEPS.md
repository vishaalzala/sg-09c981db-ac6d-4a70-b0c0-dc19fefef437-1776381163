# Upload and rollout steps for Phases 6–12 scaffold

1. Extract this zip into your project in a branch or safe copy.
2. Do not replace working business flows.
3. Run migrations one by one in Supabase SQL Editor.
4. Keep feature flags OFF initially.
5. Add admin pages gradually and test each route.
6. Only after validation, enable one module at a time.
